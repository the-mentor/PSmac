# PSmac - PowerShell for macOS

A PowerShell module designed for Apple macOS users who want to leverage native PowerShell commands for their everyday tasks. PSmac provides a collection of macOS-optimized PowerShell cmdlets that integrate seamlessly with the native macOS environment, enabling users to harness the power of PowerShell scripting while maintaining compatibility with macOS-specific workflows and conventions.

## Target Users

- macOS users transitioning from Windows PowerShell environments
- System administrators managing mixed macOS/Windows environments
- DevOps engineers seeking consistency across platforms
- Power users automating daily macOS tasks with PowerShell

## Installation

To install the module, run:

```powershell
Install-Module -Name 'PSmac' -Scope CurrentUser
```
